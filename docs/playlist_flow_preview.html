关键接口说明

MCP协议接口

{
  "method": "tools/call",
  "params": {
    "name": "self.music.play_playlist",
    "arguments": {
      "playlist_name": "流行金曲",
      "artist_name": "周杰伦"
    }
  }
}

服务器API接口

GET /playlist?query=歌单名或歌手名
Response: {
  "success": true,
  "playlist": ["歌曲1", "歌曲2", "歌曲3"]
}

音频流接口

GET /stream_pcm?song=歌曲名
Response: {
  "success": true,
  "audio_url": "/audio/xxx.mp3?token=xxx",
  "lyric_url": "/lyric/xxx.lrc?token=xxx"
}

性能优化点

  • 内存管理：及时清理音频缓冲区和歌词数据
  • TCP连接：等待连接完全关闭，避免资源泄漏
  • 任务管理：使用FreeRTOS任务替代pthread，更好的资源控制
  • 状态监控：智能处理设备状态变化，避免不必要的停止
  • 错误恢复：自动跳过失败的歌曲，保证歌单连续性
